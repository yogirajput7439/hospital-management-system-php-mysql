A **full-stack web-based Hospital Management System** built using **PHP & MySQL**.  
This project is designed to efficiently manage hospital operations such as patient records, appointments, and user roles.

---

## 🚀 Features

- 🔐 Secure Login System (Role-Based Authentication)
- 👨‍⚕️ Multi-User Roles:
  - Admin
  - Doctor
  - Receptionist
- 🧾 Patient Management (Add / View / Delete)
- 📅 Appointment Booking System
- 📊 Separate Dashboards for Each Role
- 🗂 Organized Database Structure
